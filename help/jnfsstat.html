Jnfsstat

Jnfsstat is a tabular display of NFS statistics, showing the data you can obtain from the nfsstat command.

You can show rpc statistics, connectionless and connection oriented, for client and server calls.

You can show NFS client and server statistics for NFS versions 2, 3, and 4.

You can show NFS ACL call statistics.

You can right-click on any statistic to generate a chart of that statistic.

The toolbar

The toolbar contains a small number of menus.

  • File - allows you to exit jnfsstat.
  • Update every - allows you to set how rapidly the statistics are updated.
  • Help - shows this help file, and the CDDL license that this software is licensed under.
